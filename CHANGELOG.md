# Changelog

## v2.0 (baseline documental)

- Base funcional del módulo Adulto general.
- Incorporación de módulo Oncología (v2.0 simplificado).
- Incorporación de módulo Pediátrico (v2.0 simplificado).
- Separación explícita UI (`src/pages`) y motores clínicos puros (`src/logic`).
- Documentación técnica y funcional inicial en `docs/`.

## Política de cambios

- Todo cambio clínico debe reflejarse en documentación de módulo correspondiente.
- Todo cambio de alcance debe actualizar `docs/product/scope.md`.
- Todo cambio relevante de arquitectura debe actualizar `docs/architecture.md`.
